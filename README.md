# Gushwork Assignment – Responsive Product Webpage

## Overview
This project is a responsive product webpage built using HTML, CSS, and JavaScript based on the provided Figma design. It includes a sticky header and an interactive image carousel with zoom functionality.

## Features
- Responsive design (desktop, tablet, mobile)
- Sticky header on scroll
- Image carousel with zoom effect
- Dynamic content rendering using JavaScript
- Pixel-perfect layout based on Figma
